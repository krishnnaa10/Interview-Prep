Q. Define arange and linspace.


arange    

It generates values with a specified step size between the start and stop values.
ie.. (0,10,5) = [0, 5, 10]

linspace   

It generates values with a specified number of points evenly spaced between the start and stop values.
ie.. (0,10,5) = [0, 2.5, 5.0, 7.5, 10.0]
