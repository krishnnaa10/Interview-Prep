Q. Define argmax(), argmin() & argsort() in numpy.


argmax(): Returns the index of the maximum element in an array.
ie.. np.argmax([1, 3, 2]) → 1 (index of the maximum value).

argmin(): Returns the index of the minimum element in an array.
ie.. np.argmin([1, 3, 2]) → 0 (index of the minimum value).

argsort(): Returns the indices that would sort an array.
ie.. np.argsort([1, 3, 2]) → [0, 2, 1] (indices that would sort the array).
